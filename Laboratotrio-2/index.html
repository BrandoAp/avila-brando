¿Qué es Javascript y en qué se asemeja a Java?
JavaScript es un lenguaje de programación multiplataforma orientado a objetos que se utiliza para hacer que las paginas web sean interactivas.
JavaScript y Java son simulares en algunos aspectos, pero fundamentalmente diferentes en otros. El lenguaje JavaScript se parece a Java pero no tiene el tipado estatico ni la fuerte verificacion de tipos de Java. JavaScript sigue la mayoria de la sintaxis de las expresiones de Java, convenciones de nomeclatura y construcciones de control de flujo basicas, razón por la cual se cambio el nombre de LiveScript a JavaScript.
